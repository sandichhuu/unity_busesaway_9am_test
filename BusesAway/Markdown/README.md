Buses Away - Implementation Roadmap (Markdown folder)

This folder contains step-by-step, object-by-object guidance to implement the core gameplay mechanics shown in the screenshot. Each file breaks down the concrete tasks, data models, and minimal code scaffolds needed to realize the feature end-to-end in a Unity project.

How to use
- Read files in order for a progressive build. Start with 01_passenger.md and move through subsequent objects.
- Each file contains: Objective, Data Model, Step-by-step Tasks, Acceptance Criteria, and Code Skeletons.
- Use the code blocks as starting points; adapt namespaces and project structure to your codebase.

Note
- This is a blueprint. If your project uses a specific architecture (MVVM, ECS, MCP, etc.), adjust the patterns accordingly but keep the core logic intact.
