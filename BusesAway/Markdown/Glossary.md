# Glossary

- Passenger: An individual unit in a color group waiting to board a bus.
- Color: Visual category used for matching boarding rules (Red, Green, Yellow, Blue, Purple).
- Lane: One of three queues at the bottom where passengers accumulate.
- Waiting Room: Interim area before boarding; bus picks up matching-color passengers here.
- Bus: Vehicle with a color and capacity; boards passengers of the same color.
- Level: A stage in the game with a progression bar.
- Spawner: Entity responsible for generating waves of passengers and colors.
- Game Controller: Orchestrator that wires together all components per level.
- State Machine: Governs transitions between core gameplay phases.
